# WhatsApp_clone
Création d'un clone whatsApp.

## Pour lancer l'application
`cd Front && npm i && npm start`  
`cd Back && nodemon server.js ` 

## Technologies utilisées
* MongoDB
* Express
* React
* Node
* CSS3
* Material UI
* Firebase
* Pusher_socket
* Axios
